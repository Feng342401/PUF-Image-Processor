# PUF Image Processing and Statistical Analysis Toolkit

## Overview

This repository provides a fully reproducible pipeline for image-based Physically Unclonable Function (PUF) analysis.

The toolkit supports:

- Image preprocessing and binarization
- Bit uniformity computation
- Hamming distance computation
- Batch statistical evaluation
- Security characterization of PUF-based systems

---

## Key Features

- Fully deterministic pipeline
- Automatic parameter optimization
- Reproducible randomness calibration
- Batch-level statistical evaluation
- Standardized evaluation metrics for PUF security
